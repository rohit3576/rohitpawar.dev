AI Mental Health Analyzer

Overview

Built an NLP-based system to analyze user text and detect sentiment and stress levels. Real-time predictions with a full-stack ML application.

The Problem

Mental health issues are often undetected due to lack of accessible screening tools. Traditional methods require clinical visits, creating barriers for early intervention.

The Solution

An AI-powered web application that analyzes text input to detect emotional states, stress levels, and provides actionable insights. Built with NLP models and deployed as a user-friendly interface.

Technical Implementation

  • Used BERT-based model for sentiment analysis
  • Implemented REST API with Flask for model serving
  • React frontend with real-time predictions
  • Deployed on cloud with CI/CD pipeline

Impact & Results

Achieved 85% accuracy in stress detection, deployed for pilot testing with 500+ users, and received positive feedback on early intervention capabilities.
